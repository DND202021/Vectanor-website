# Vectanor-website

Repo centralisé pour la gestion des sites web du Groupe Vectanor.

## Structure

```
/dimonoff      — dimonoff.com (Smart Lighting, Smart Cities, IoT)
/spatium       — spatium-iot.com (Intelligent Parking Management)
/vectanor      — vectanor.com (Holding / Group)
/stratys       — stratys.ca (Defense Consortium)
/amotus        — amotus.com (IoT Engineering, Electronics Design)
```

Chaque dossier de site contient :

- `STATUS.md` — Statut actuel, stack technique, projets actifs
- `mu-plugins/` — Fichiers PHP mu-plugins deployés sur le site
- `docs/` — Plans, rapports, documents de référence
- `audits/` — Audits SEO, AEO/GEO, rapports de validation

## Conventions

- Tout changement à un mu-plugin doit d'abord être commité ici avant déploiement
- Les fichiers `.docx` de recommandations IT sont dans `docs/`
- Les audits sont horodatés dans le nom de fichier ou dans le contenu
