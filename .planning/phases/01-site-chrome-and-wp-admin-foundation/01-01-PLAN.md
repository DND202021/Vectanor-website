<task type="checkpoint:human-action" gate="blocking">
  <name>Task 1: Create FR/EN/ES Navigation Menus and Build FR Header Template</name>
  <action>
This task creates the navigation menus and the French header template in WP Admin. Follow these steps exactly:

**Part A: Create Navigation Menus**

1. Go to WP Admin > Appearance > Menus
2. Create menu "Menu Principal (FR)":
   - Add these pages as menu items: Accueil (page 19), Notre vision (page 20), Nos divisions (page 21), Contact (page 25)
   - Under "Menu Settings" at the bottom, check the menu location for French (Polylang creates per-language menu locations -- look for the one marked FR or "Primary - French")
   - Save Menu
3. Create menu "Main Menu (EN)":
   - Add Custom Links for each item (since EN pages may not all exist yet):
     - Home -> /en/
     - Our Vision -> /en/notre-vision/ (or the EN slug if translated)
     - Our Divisions -> /en/divisions/
     - Contact -> /en/contact/
   - Assign to the EN primary menu location
   - Save Menu
4. Create menu "Menu Principal (ES)":
   - Add Custom Links:
     - Inicio -> /es/
     - Nuestra vision -> /es/notre-vision/ (or the ES slug if translated)
     - Nuestras divisiones -> /es/divisions/
     - Contacto -> /es/contact/
   - Assign to the ES primary menu location
   - Save Menu

**Part B: Build FR Header Template**

5. Go to WP Admin > Templates > Theme Builder (or Elementor > Templates > Theme Builder)
6. Click "Add New" > select "Header"
7. Name it "Header - FR" (or "Header")
8. Choose a blank/empty template (do NOT use a pre-built header template)
9. Build the header with this structure:
   - Add a Container (flexbox row, justify: space-between, align: center)
   - Container settings:
     - Content Width: Full Width
     - Padding: 15px top/bottom, 30px left/right
     - Background: #0F172A (near black)
   - Inside the container, add three widgets in this order:

   a) **Site Logo widget** (left):
      - Link to: Site URL (homepage)
      - Max width: ~150px
      - Custom CSS (if needed to make it white): `selector img { filter: brightness(0) invert(1); }`

   b) **Nav Menu widget** (center/right):
      - Select menu: "Menu Principal (FR)"
      - Layout: Horizontal
      - Breakpoint: Tablet (this enables hamburger on mobile/tablet)
      - Toggle button: Hamburger
      - Typography: Montserrat, 14px, weight 600
      - Text color: #FFFFFF (white)
      - Text hover color: #3B82F6 (royal blue)

   c) **Language Switcher widget** (far right):
      - This is provided by Connect Polylang for Elementor (look in "General Elements" or search "Language Switcher")
      - Layout: Horizontal
      - Show flags: Yes
      - Show name: No
      - Show code: Yes (displays FR, EN, ES)

10. Set Display Condition: "Entire Site"
11. Click Publish

**IMPORTANT:** Do NOT set display conditions on translated templates later -- only on this FR template. Connect Polylang for Elementor handles routing automatically.
  </action>

</task>

<task type="checkpoint:human-action" gate="blocking">
  <name>Task 2: Create EN and ES Header Translations</name>
  <action>
This task creates the English and Spanish header translations using Polylang's template translation feature.

**Part A: Create EN Header Translation**

1. Go to WP Admin > Templates > Saved Templates (or Elementor > Templates > Saved Templates)
2. Find the FR header template you just created
3. In the Polylang language columns, click the "+" icon in the EN column to create an English translation
4. Elementor editor opens with a copy of the FR header
5. Click the Nav Menu widget and change the selected menu from "Menu Principal (FR)" to "Main Menu (EN)"
6. The Language Switcher widget does NOT need changes -- it auto-adapts to show current language
7. Do NOT set any display conditions (Connect Polylang for Elementor handles this)
8. Click Publish

**Part B: Create ES Header Translation**

9. Go back to Templates > Saved Templates
10. Find the FR header template again
11. Click the "+" icon in the ES column
12. In the editor, change the Nav Menu widget's selected menu to "Menu Principal (ES)"
13. Do NOT set display conditions
14. Click Publish

**Part C: Verify Polylang Page Stubs**

Before testing language switching, verify EN/ES homepage stubs are properly set up:

15. Go to WP Admin > Pages
16. Find page 30 (Accueil - English) -- confirm it shows "EN" in the Polylang language column
17. Find page 31 (Accueil - Espanol) -- confirm it shows "ES" in the Polylang language column
18. Confirm pages 30 and 31 are linked as translations of page 19 (click the flag icons to check translation links)
19. If pages 30/31 are NOT linked to page 19 as translations, click the "pencil" icon in the appropriate language column on page 19's row to link them

**WARNING:** If the Nav Menu in translated headers still shows French labels (Accueil, Notre vision...), you forgot to change the menu selection in step 5 or 12. Go back and edit.
