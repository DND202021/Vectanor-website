<task type="checkpoint:human-action" gate="blocking">
  <name>Task 1: Build FR Footer Template in Theme Builder</name>
  <action>
Create the French footer template in Elementor Pro Theme Builder.

1. Go to WP Admin > Templates > Theme Builder (or Elementor > Templates > Theme Builder)
2. Click "Add New" > select "Footer"
3. Name it "Footer - FR" (or "Footer")
4. Choose a blank/empty template
5. Build the footer with this structure:

   - Add a Container (flexbox column, justify: center, align: center)
   - Container settings:
     - Content Width: Full Width
     - Padding: 40px top/bottom, 30px left/right
     - Background: #0F172A (near black, same as header)

   Inside the container, add these widgets vertically:

   a) **Image widget** (Vectanor logo):
      - Upload or select the Vectanor logo from Media Library (vectanor_logo_concept_3.svg, Media ID 9)
      - Max width: ~120px
      - Alignment: Center
      - If logo appears dark on dark background, add Custom CSS: `selector img { filter: brightness(0) invert(1); }`

   b) **Text Editor widget** (copyright):
      - Text: "(c) 2026 Groupe Vectanor. Tous droits reserves."
      - Typography: Inter, 13px, color #475569
      - Alignment: Center
      - Margin top: 15px

   c) **Text Editor widget** (legal links):
      - Text (as HTML): `<a href="#">Politique de confidentialite</a>  |  <a href="/contact/">Contact</a>`
      - Link typography: Inter, 13px, color #94A3B8
      - Link hover color: #3B82F6 (royal blue)
      - Alignment: Center
      - Margin top: 10px
      - NOTE: The privacy policy link uses "#" as a placeholder -- it will be updated in Phase 3 when legal pages are created

6. Set Display Condition: "Entire Site"
7. Click Publish

**Styling reference:**
- Background: #0F172A (matches header)
- Logo: white/inverted, ~120px width
- Copyright text: Inter 13px, color #475569 (muted slate)
- Link text: Inter 13px, color #94A3B8 (light slate), hover #3B82F6 (royal blue)
- Overall: Centered column, clean and minimal
